# 🛠️ Real Classic — Мод для Tanki Classic

> **Версия:** 0.7.5
> **Платформа:** Браузер Chrome

## Уведомления о новых версиях в Telegram: https://t.me/tanki_classic_real

## Поддержать: https://web.tribute.tg/d/HyV

---

## 🚀 Установка (в браузере)

1.  Откройте страницу расширений браузера: `chrome://extensions/`
2.  Включите режим разработчика в правом верхнем углу.
3.  Нажмите кнопку **«Загрузить распакованное расширение»**.
4.  Выберите папку с файлами мода (должен быть файл `manifest.json`).

## 🚀 Установка (в клиент)
1.  Скачать main.js (достаточно один раз скачать, подойдет на любые версии мода, начиная с 0.7.4 версии)
2.  Скачать real-classic-v0.7.5.zip
3.  Распаковать real-classic-v0.7.5.zip (без вложенных папок, чтобы сразу по открытии папки появились файлы мода)
4.  Открыть папку C:\Users\<Ваш пользователь>\AppData\Local\Programs\Tanki Online\resources\app.asar.unpacked
5.  Заменить в этой папке main.js
6. Перенести туда же папку с расширением
7. Можно запускать (первый запуск - вылет (установка расширения), со второго уже можно заходить в классику)

## Как запустить мод на мобильных устройствах?

Для Android: достаточно любого браузера, который может устанавливать расширения (например, Kivi Browser), просто скачиваем .zip архив расширения по ссылке и он подойдет для установки
Для iOS: аналогично, Orion Browser имеет возможность установки расширения, также подойдет скаченный .zip архив для установки
